What FleetMagnify Reports Look Like

Every report in FleetMagnify is derived from your actual telematics and fuel card data — not estimates, not averages, not industry benchmarks. Here is exactly what each module outputs and how the numbers are calculated.

Report 1

Cost Per KM Report

Fields output
Asset Name Period Start Period End KM Driven Litres Used Fuel Cost/km Depreciation/km Maintenance/km Servicing/km RUC/km Total Cost/km Cost Per Tonne-km
How calculated

Fill-to-fill methodology — actual fuel purchases paired with real odometer readings. Depreciation calculated from current value, end-of-life value, and expected life km. RUC calculated from vehicle GVM using Waka Kotahi rate tables.

Report 2

Job Cost Report

Fields output
Job Name Asset Name Days Worked Fuel Cost True Idle Cost Depreciation Servicing Maintenance Asset Total Grand Total Job Cost Cost Per Operating Hour Cost Per Tonne
How calculated

Operating hours from telematics matched to job dates. Fuel cost from fuel card transactions. Depreciation, servicing and maintenance calculated from asset profiles. Idle cost separated from operating cost using telematics idle time data.

Report 3

Idle Cost Report

Fields output
Asset Name Total Engine Hours Operating Hours Idle Hours Idle % Idle Fuel Cost Idle Cost as % of Total Cost
How calculated

Engine hours and idle minutes from telematics. Idle fuel cost calculated using calibrated idle burn rate per asset, derived from fill-to-fill regression analysis against real fuel card data.

Report 4

Trip Report

Fields output
Asset Name Trip Start Trip End Distance (km) Total Fuel Cost Depreciation Cost Maintenance Cost Servicing Cost Total Trip Cost Cost Per KM Payload (tonnes) Cost Per Tonne-KM Emissions (kg CO2-e)
How calculated

Distance from telematics odometer readings. Fuel cost from fill-to-fill rate applied to trip distance. Emissions from actual fuel consumption × MfE diesel emission factor (2.680680605 kg CO2-e/L).

Report 5

Emissions Report

Fields output
Asset Name Period Litres Used KM Driven Total Emissions (kg CO2-e) Emissions Per KM Payload (tonnes) Emissions Per Tonne-KM
How calculated

Actual fuel consumption from fill-to-fill analysis multiplied by NZ Ministry for the Environment emission factor for diesel (2.680680605 kg CO2-e per litre). All figures are data-derived, not estimated.
