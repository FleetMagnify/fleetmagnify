📊 What it shows
  • ✓ Daily fuel consumption chart — operating fuel vs idle fuel, stacked per day
  • ✓ Travelling fuel rate (L/hr under load) and idle fuel rate (L/hr at idle)
  • ✓ Total fuel cost for the selected period based on your bulk diesel price
  • ✓ Idle fuel cost as a dollar figure and percentage of total fuel spend

🔢 How it calculates
  • ✓ Total litres consumed comes from your OEM telematics ECU — not from fuel invoices
  • ✓ Idle fuel = idle hours × idle burn rate (set per asset, or platform default by machine type)
  • ✓ Operating fuel = total litres minus idle fuel
  • ✓ Fuel cost = litres × bulk diesel price per litre (set in Settings — update when your tank price changes)
💡
Idle burn rate defaults apply by machine type if not set on the individual asset — Excavator 4.0 L/hr, Dump Truck 5.5 L/hr, Wheel Loader 4.5 L/hr and so on. Set the actual rate on each asset for more accurate calculations.

🔢 How fill-to-fill works
  • ✓ Fill 1 establishes the starting odometer — excluded from calculations (no prior fill to compare)
  • ✓ Fill 2 litres ÷ km driven since Fill 1 = L/100km for that specific period
  • ✓ Each subsequent fill gives an accurate consumption rate for that driving period
  • ✓ eRoad odometer readings closest to each fill date are used for maximum precision
  • ✓ BP card number on each asset links every transaction to the right vehicle automatically

🔧 Fleet Management

Set up once. Runs itself.

Add your assets, connect your APIs, set your diesel price. Every calculation across every module flows from these three steps.

Fleet Management
🚧
Assets
Your central fleet register — the foundation of every cost calculation across the platform.
⚙️ Machinery asset settings
  • ✓ Asset name (must match OEM telematics platform exactly), type, make, model, year
  • ✓ Current value and end of life value — used for depreciation per engine hour
  • ✓ Expected life hours, service cost per hour, estimated annual repair cost
  • ✓ Available hours per week and target utilisation percentage
  • ✓ Idle burn rate in L/hr — defaults apply by machine type if not set
🚛 Truck asset settings
  • ✓ Asset name (must match eRoad account exactly), type, make, model, registration
  • ✓ Current value and end of life value — used for depreciation per km
  • ✓ Expected life km, service cost per km, estimated annual repair cost
  • ✓ Estimated annual km and current odometer reading
  • ✓ BP card number — links every BP transaction to the right vehicle automatically
📥
Excel import available. Download the Asset Import Template from the Upload Data page, complete the Machinery and Trucks sheets, and upload to add your entire fleet at once.
